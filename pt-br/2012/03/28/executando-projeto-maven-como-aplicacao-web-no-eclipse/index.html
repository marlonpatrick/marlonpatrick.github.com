Executando projeto Maven como aplicação Web no Eclipse

28/03/2012 em Maven, Eclipse

Esses dias estava configurando meu Eclipse para trabalhar com Demoiselle Framework e como esse framework é totalmente gerenciado pelo Maven um dos plugins necessários é o m2eclipse para fazer integração entre Eclipse e Maven.

Até aí tudo bem, você pode abrir o Eclipse ir no menu Help > Install New Software e o usar o update site http://download.eclipse.org/technology/m2e/releases para instalar o m2eclipse. Com isso, eu pude usar um dos arquétipos do Demoiselle para baixar uma aplicação já totalmente configurada.

Porém, um problema que eu tive foi o de fazer esse projeto executar como uma aplicação web via Eclipse e aí depois de muito fuçar, vi que precisaria de um outro plugin para fazer essa integração maven/eclipse/aplicação web funcionar, é o m2e-extras. Para isso, basta você fazer o mesmo procedimento acima e usar o update site http://m2eclipse.sonatype.org/sites/m2e-extras.

Só para garantir, vá no menu Project > Properties > Project Facets e marque a opção Dynamic Web Module. Depois disso, é só clicar com o botão direito no projeto ir na opção Run as > Run on Server e pronto, projeto Maven executando como uma aplicação web normalmente!

Update:No caso de haver problemas de compatibilidade com a versão do m2e-extras, então, será necessário usar o update site http://m2eclipse.sonatype.org/sites/m2e-webby/ que suporta a versão mais recente do Maven.
